## CHECK(检查终止)约束
* 创建一个id_p必须大于0
<pre><code>
CREATE TABLE a
(
id_p int NOT NULL CHECK (id_p>0),
LastName varchar(255) NOT NULL,
FirstName varchar(255)
)
</code></pre>

* 命名CHECK约束
<pre><code>
ALTER TABLE a ADD CONSTRAINT ll_ii CHECK (id_p>0 AND FirstName='zeshuai')
</code></pre>

* 删除CHECK约束
<pre><code>
ALTER TABLE a DROP CONSTRAINT ll_ii
</code></pre>
